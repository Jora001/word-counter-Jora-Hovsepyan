# Text Counter

This is my own one-page application that counts letters, words, sentences, and other text metrics.  
I created this project for **TUMO Labs** as part of the JavaScript station projects. I also added **bonus features** like a random background generator.

## Features
- Count letters
- Count characters (with spaces)
- Count words
- Count sentences
- Count paragraphs
- Find the longest word
- Calculate average word length
- **Bonus:** Generate a random background (added by me)

## Usage

1. Type or paste text into the textarea.
2. Click **Count now** to see the results.
3. Optionally, enable **Live update** for automatic counting as you type.
4. Click **Change background** to generate a random background.
